# Exercice 5 : Conditions et Boucles

## Objectif
Maîtriser les structures de contrôle : if/else, switch, for, while, foreach.

## Instructions

### Partie 1 : Conditions
1. Créez une variable `$note` avec une valeur entre 0 et 20
2. Affichez la mention correspondante :
   - >= 16 : "Très bien"
   - >= 14 : "Bien"
   - >= 12 : "Assez bien"
   - >= 10 : "Passable"
   - < 10 : "Insuffisant"

### Partie 2 : Boucles
3. Avec une boucle `for`, affichez les nombres de 1 à 10
4. Avec une boucle `while`, affichez la table de multiplication de 7
5. Avec une boucle `foreach`, parcourez un tableau de prénoms

## Aide
- `if`, `elseif`, `else` pour les conditions
- `for ($i = 0; $i < 10; $i++)` pour une boucle for
- `foreach ($tableau as $element)` pour parcourir un tableau
