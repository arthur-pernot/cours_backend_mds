# Exercice 6 : Les Tableaux

## Objectif
Manipuler les tableaux indexés, associatifs et multidimensionnels en PHP.

## Instructions

### Partie 1 : Tableaux associatifs
1. Créez un tableau associatif `$etudiant` avec les clés : nom, prenom, age, email
2. Affichez chaque information
3. Ajoutez une clé "ville" au tableau
4. Modifiez l'âge de l'étudiant

### Partie 2 : Tableaux multidimensionnels
5. Créez un tableau `$classe` contenant 3 étudiants (chacun avec nom, prenom, note)
6. Affichez les informations de tous les étudiants
7. Calculez et affichez la moyenne de la classe

## Aide
- Tableau associatif : `$tab = ["cle" => "valeur"];`
- Accéder à une valeur : `$tab["cle"]`
- Ajouter un élément : `$tab["nouvelleCle"] = "valeur";`
- Parcourir : `foreach ($tab as $cle => $valeur)`
