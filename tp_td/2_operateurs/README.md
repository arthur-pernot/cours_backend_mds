# Exercice 2 : Les Opérateurs

## Objectif
Apprendre à utiliser les opérateurs arithmétiques, de comparaison et logiques en PHP.

## Instructions
1. Créez deux variables `$a = 10` et `$b = 3`
2. Affichez le résultat de : addition, soustraction, multiplication, division, modulo
3. Testez les opérateurs de comparaison (`==`, `!=`, `>`, `<`, `>=`, `<=`)
4. Créez deux variables booléennes et testez les opérateurs logiques (`&&`, `||`, `!`)
5. Utilisez les opérateurs d'incrémentation (`++`, `--`)

## Résultat attendu
Une page affichant les résultats de toutes les opérations.

## Aide
- Modulo : `$a % $b` (reste de la division)
- Comparaison : retourne `true` ou `false`
- Pour afficher un booléen : `var_dump($result);`
