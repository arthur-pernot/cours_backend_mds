# Exercice 1 : Les Variables

## Objectif
Apprendre à déclarer et manipuler des variables en PHP.

## Instructions
1. Créez une variable `$nom` contenant votre prénom
2. Créez une variable `$age` contenant votre âge (nombre entier)
3. Créez une variable `$taille` contenant votre taille en mètres (nombre décimal)
4. Créez une variable `$estEtudiant` contenant `true`
5. Affichez chaque variable avec `echo`
6. Affichez le type de chaque variable avec `gettype()`

## Résultat attendu
Une page affichant vos informations personnelles et le type de chaque variable.

## Aide
- Pour déclarer une variable : `$maVariable = "valeur";`
- Pour afficher : `echo $maVariable;`
- Pour connaître le type : `gettype($maVariable);`
